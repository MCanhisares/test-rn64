# Test app RN 0.64

## Requirements

- [Yarn](https://classic.yarnpkg.com/en/docs/install/)
- [React Native environment](https://reactnative.dev/docs/environment-setup)

## Running the app

## iOS
```sh
yarn

cd ios && pod install

yarn ios:run:dev
```

## Android

```sh
yarn

yarn android:run:dev
```
